# blockchainos

## directories:

blockchain - source code for the blockchain
* blockchain/include
* blockchain/lib
* blockchain/obj
* blockchain/src
* Makefile

node - client code

## dependencies
OpenSSL
LibEv

Install Dependencies:
```
sudo apt install libssl-dev
sudo apt install libev-dev
```
